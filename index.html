Setup Guide:
1. GitHub → Settings → Developer settings → Tokens → Fine-grained
2. Select repo, grant Contents: Read and write
3. File Path = data.json (데이터만 동기화, HTML 코드와 분리)
4. Paste token, enter owner/repo, click Test
